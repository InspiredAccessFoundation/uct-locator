Basic Background

Basic Background: Web Development & Command Line

A fundamental knowledge of basic web development and the command line will be necessary to start working on this project.

You should be at least basically familiar with three languages:

  • HTML
  • CSS
  • JavaScript

For a very quick half-hour crash course on all three topics, check out this video.

The command line/terminal will also be used - for a quick interactive introduction, play this game.

JavaScript Concepts

There are some JavaScript concepts that are used throughout the codebase - some of these are not covered by the basic tutorials. It can be helpful to try to understand these concepts before diving into the code.

  • const: Several variables are declared using the const keyword, instead of var or let.
  • Objects: Objects are the most basic form of data in JavaScript. They are used heavily throughout the codebase.
  • JSON data: This is the form of the data sent between the front-end and back-end of the product.
  • Template literals: This is another way to create strings in JavaScript.
  • Arrow Functions: This is another way to write functions in JavaScript. These are often used as callbacks.
  • import: This is one way to include libraries of other code into a JavaScript file.
  • Module exports: The other side of the import, this is how code is exported from a JavaScript file.
  • Promises: These are a fairly complex way to handle asynchronicity in JavaScript. Most of the code does not work with promises directly, and uses async/await instead.
  • async/await: This paradigm makes promises easier to write. async and await are typically used when retrieving data from the database, or waiting for a response request.
  • map: This is a way to iterate over an array.

Many of these concepts are quite advanced, so don't worry if you don't fully grasp all of them!
